# Config

Este directorio contiene archivos de configuración de la aplicación.

## Estructura

```
config/
├── constants.ts      # Constantes de la aplicación
├── api.ts            # URLs y configuración de APIs
└── site.ts           # Metadata del sitio
```

## Uso

```typescript
import { API_ENDPOINTS, SITE_CONFIG } from '@/config';
```
